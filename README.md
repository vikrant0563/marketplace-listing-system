# MERN Marketplace Listing System

A full-stack marketplace web application built using:

- React (Vite)
- Node.js
- Express.js
- MongoDB
- Cloudinary
- Cookie-based Authentication

## Features
- User Authentication
- Create, Update, Delete Listings
- Search & Filters
- Category Management
- Image Upload (Cloudinary)

## Setup

### Backend
cd backend
npm install
npm run dev

### Frontend
cd frontend
npm install
npm run dev
